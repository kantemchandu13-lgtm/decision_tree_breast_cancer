# Decision Tree Classification on Breast Cancer Data

This project demonstrates how a Decision Tree Classifier can be used to classify breast cancer diagnoses as benign or malignant.

## Main focus
The notebook investigates how changing tree depth affects:
- accuracy
- recall
- overfitting
- interpretability

## Files
- `decision_tree_breast_cancer.ipynb` - main notebook
- `breast_cancer_data.csv` - breast cancer dataset file
- `requirements` - requirements file

## How to run
1. Install required libraries
2. Place the CSV dataset in the same folder
3. Open the notebook in Jupyter
4. Run all cells

## Dataset
The dataset contains breast cancer measurements and a diagnosis column:
- M = malignant
- B = benign
